Dúvidas?

Perguntas frequentes

É mesmo pagamento único? +
Sim. Pagas 670 MZN uma única vez e o app é teu para sempre. Não há mensalidades nem cobranças escondidas.
Como recebo o app depois de pagar? +
Logo após confirmares o pagamento, recebes um link para baixar e instalar o app no teu telemóvel Android. É rápido e simples.
Os meus dados financeiros ficam seguros? +
Sim. Os teus dados ficam guardados apenas no teu próprio telemóvel — não são enviados para servidores nem partilhados com ninguém.
Funciona sem internet? +
Sim. Depois de instalado, registas as tuas transações e metas mesmo sem ligação à internet. Tudo funciona offline.
Qual a diferença entre conta individual e familiar? +
A conta individual é só para ti. A familiar permite-te organizar as finanças da casa toda num só lugar, com um nome de família próprio. Escolhes ao criar a conta.
